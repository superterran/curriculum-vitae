This is a placeholder Word document template for Pandoc.

To create a custom styled template:
1. Generate a resume: npm run build:all
2. Open dist/resume.docx in Microsoft Word
3. Modify styles (Heading 1, Heading 2, Normal, etc.)
4. Save as templates/reference.docx
5. Future builds will use your custom styles

For more info: https://pandoc.org/MANUAL.html#option--reference-doc
